📋 Менеджер заметок
📘 О программе
Менеджер заметок — это консольное приложение для создания, управления и организации текстовых заметок. Программа позволяет 
эффективно хранить и обрабатывать информацию в формате текстовых файлов.
________________________________________
🎯 Основные возможности
•	Создание заметок — быстрое добавление новых записей
•	Просмотр заметок — удобный доступ к сохраненным данным
•	Редактирование — изменение существующих заметок
•	Удаление — безопасное удаление ненужных записей
•	Список заметок — просмотр всех сохраненных заметок с сортировкой
________________________________________
🔧 Технические требования
•	Python версии 3.x
•	Права доступа — возможность записи в директорию запуска
________________________________________
📋 Установка и запуск
1.	Сохраните код в файл с расширением main_1.py
2.	Откройте терминал в директории с файлом
3.	Запустите программу
📂 Использование
После запуска вы увидите главное меню с опциями:
•	1. Создать заметку — добавление новой записи
•	2. Читать заметку — просмотр существующей записи
•	3. Редактировать заметку — изменение содержимого
•	4. Удалить заметку — удаление записи
•	5. Показать все заметки — список всех заметок
•	6. Выход — завершение работы
________________________________________
🛡️ Особенности безопасности
•	Валидация ввода — проверка на корректность названий
•	Защита от ошибок — обработка некорректных действий
•	Фильтрация символов — очистка названий от опасных символов
________________________________________
📊 Ограничения
•	Длина названия — максимум 100 символов
•	Формат хранения — заметки сохраняются в текущей директории как .txt файлы
•	Подтверждение — требуется подтверждение при удалении заметок
________________________________________
🔍 Обработка ошибок
Программа автоматически обрабатывает:
•	Ошибки доступа — проблемы с правами записи
•	Некорректные данные — ошибочный ввод
•	Прерывание — корректное завершение при Ctrl+C
•	Отсутствие файлов — попытки работы с несуществующими заметками
